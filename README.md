# Belu Francia · Fisioterapia & Osteopatía en Cartagena

Web profesional de fisioterapia y osteopatía para **Belu Francia**, Cartagena (Murcia).  
Desarrollada por [Ruipérez Studio](https://ruiperezstudio.es).

## Stack

- HTML5 semántico
- CSS3 con custom properties (sin preprocesadores)
- JavaScript vanilla (IIFE, sin módulos)
- GSAP 3.12.5 + ScrollTrigger (local)
- Google Fonts (Fraunces + Inter)
